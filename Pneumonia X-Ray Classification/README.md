# Pneumonia X-Ray Classification

## Project Overview
This project uses Deep Learning techniques to classify chest X-ray images as Pneumonia or Normal. The goal is to assist healthcare professionals by providing an AI-powered screening tool.

## Objectives
- Preprocess chest X-ray images.
- Build and compare deep learning models.
- Detect pneumonia from X-ray scans.
## Dataset
The dataset can be uploaded from:
Kaggle : https://www.kaggle.com/code/pkdarabi/diagnosis-of-pneumonia-by-cnn-classifier/input

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Workflow
1. Data Loading
2. Image Preprocessing
3. Data Augmentation
4. Model Training
5. Performance Evaluation
6. Prediction

## Results
The trained model achieved strong classification performance and successfully identified pneumonia cases from chest X-ray images.
