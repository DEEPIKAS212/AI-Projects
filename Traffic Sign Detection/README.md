# Traffic Sign Detection

## Project Overview
This project focuses on recognizing traffic signs from images using Computer Vision and Deep Learning techniques. The system helps improve intelligent transportation systems and road safety applications.

## Objectives
- Perform image preprocessing.
- Train a model to classify traffic signs.
- Evaluate model performance.
## Dataset
The dataset can be downloaded from:
Kaggle : https://www.kaggle.com/code/notsau/german-traffic-sign-classification-with-dl/input
## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Workflow
1. Data Collection
2. Image Preprocessing
3. Data Augmentation
4. Model Training
5. Evaluation
6. Prediction

## Results
The model successfully classifies different traffic sign categories with high accuracy and demonstrates the effectiveness of deep learning for traffic sign recognition.
