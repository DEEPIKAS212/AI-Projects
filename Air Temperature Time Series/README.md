# Air Temperature Time Series Forecasting

## Project Overview
This project forecasts future air temperature trends using historical climate data and Time Series Analysis techniques. Multiple forecasting models were developed and compared to identify the best-performing approach.

## Objectives
- Analyze historical temperature data.
- Forecast future monthly mean temperatures.
- Compare multiple forecasting techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Statsmodels
- TensorFlow / Keras

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Stationarity Testing
4. Feature Engineering
5. Model Training
6. Forecasting and Evaluation

## Models Used
- ARIMA
- SARIMA
- Random Forest
- XGBoost
- LSTM

## Results
The project successfully forecasted future air temperature trends and compared multiple models using evaluation metrics such as RMSE and MAE.
